# Flutter Calculator 📱

A beautiful and simple calculator application built with Flutter. This is my first project as a beginner in Flutter development! 🚀

## ✨ Features

- **Basic Arithmetic**: Addition, subtraction, multiplication, division, and modulus.
- **Clean UI**: A modern, dark-themed user interface with orange and cyan accents.
- **Responsive Design**: Works smoothly on different screen sizes.
- **History/Expression View**: Shows the current expression being typed.
- **Backspace & Clear**: Easy correction of mistakes.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **Language**: [Dart](https://dart.dev/)

## 🚀 Getting Started

To run this project locally:

1.  **Prerequisites**: Make sure you have Flutter installed.
2.  **Clone the repo**:
    ```bash
    git clone https://github.com/pratham-prog861/flutter-calculator-app.git
    ```
3.  **Install dependencies**:
    ```bash
    flutter pub get
    ```
4.  **Run the app**:
    ```bash
    flutter run
    ```
